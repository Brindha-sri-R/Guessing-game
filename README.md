# 🎯 Guessing Game

An interactive number guessing game developed using HTML, CSS, and JavaScript.  
This project allows users to guess a randomly generated number between 1 and 100 and provides instant feedback based on the entered value.

## ✨ Features

- Random number generation 🎲
- User input validation 🔢
- Instant result feedback ⚡
- Interactive gameplay 🎮
- Responsive and attractive UI 🎨
- Beginner-friendly JavaScript project 💻

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
